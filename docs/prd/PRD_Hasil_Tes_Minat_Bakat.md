# Product Requirements Document (PRD)

## Feature: Hasil Tes Minat & Bakat

### Overview
Halaman hasil tes menampilkan analisis lengkap berdasarkan jawaban peserta setelah menyelesaikan tes.

### Objectives
- Menampilkan hasil tes.
- Membantu memahami potensi diri.
- Memberikan rekomendasi jurusan dan profesi.
- Mendukung unduh PDF.

### User Story
Sebagai peserta tes, saya ingin melihat hasil analisis minat dan bakat saya sehingga saya memahami potensi diri dan memperoleh rekomendasi yang sesuai.

### Main Flow
1. Selesaikan tes.
2. Sistem menghitung skor.
3. Menentukan karakter dominan.
4. Menentukan gaya belajar.
5. Menampilkan rekomendasi jurusan dan profesi.
6. Menampilkan halaman hasil.

### Layout
- Header
- Ringkasan Hasil
- Grafik Karakter
- Karakter Terkuat
- Karakter yang Perlu Dikembangkan
- Analisis Kepribadian
- Gaya Belajar
- Rekomendasi Jurusan
- Rekomendasi Profesi
- Kesimpulan
- Saran Pengembangan
- Action (Download PDF, Bagikan, Ulangi Tes)

### Functional Requirements
- Tampilkan hasil setelah tes selesai.
- Hitung skor otomatis.
- Tampilkan grafik.
- Tampilkan rekomendasi.
- Unduh PDF.

### Non Functional Requirements
- Responsive.
- Aman.
- Cepat.
- Mobile friendly.

### Acceptance Criteria
- Hasil tampil.
- Grafik benar.
- PDF dapat diunduh.
- Berjalan di desktop dan mobile.
